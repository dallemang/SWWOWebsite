About       The Data       The Queries

RDF includes all the XML data types, so you can work with data just like you do in XML-based languages. Here, we see how we can work with dates.

You can download the dataset for these queries here.

You can cross-reference the queries to the book by query number or name.

Query ch6.16 When did Giants die

We can find the dates of death of the actors in the film, Giant.

Query ch6.17 Giants who lived five years more

We can filter that to find actors who lived five years past the release of the film. We can compare any datatypes that XML considers comparable; so we can compare one date to another.
