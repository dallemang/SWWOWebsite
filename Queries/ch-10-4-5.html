  • ClassificationOfReports. This dataset is derived using the "Supplemental" query based on Query ch10.2

Query ch10.4 Incidents at location

This query sorts the incidents by location, using the geographic information about counties from dbpedia (latitude, longitude), and reporting each case based on a location in the country. The locations are approximate, since we only have county information for each one.

Query ch10.5 Counting complaint types by location

The previous query isn't very useful, since there is one result for each case, and there are thousands of cases. This query aggregates the answers, counting the number of each type of case per county, and cross referencing that number to the approximate location.
